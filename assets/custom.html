OkiDoki Documentation

Simple, fast documentation generator that turns your markdown files into beautiful docs.

Quick Start Documentation

Get running in 30 seconds

npm install -g okidoki
mkdir mydocs && cd mydocs
okidoki init
okidoki generate
npx serve dist

# Server running at http://localhost:3000

Simple Configuration

okidoki.yaml

site:
title: "My API Docs"
description: "Documentation for my API"
theme:
    light: "fantasy"
    dark: "aqua"

globals:
    version: "1.0.0"
    api_url: "https://api.example.com"

sidebars.yaml

menu:
- title: Getting Started
    document: /start.md
- title: API Reference
    document: /api.md
- title: Examples
    document: /examples.md
